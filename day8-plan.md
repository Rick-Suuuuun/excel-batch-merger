# Day 8 项目计划：Excel 批量合并、去重与来源标记工具

## 项目目标

开发一个可以批量合并多个 Excel 文件的小工具。

## 核心功能

1. 选择一个输入文件夹
2. 批量读取文件夹内所有 .xlsx 文件
3. 默认读取每个文件的第一个工作表
4. 检查表头是否一致
5. 将所有数据纵向合并
6. 添加“来源文件”列
7. 支持按一列或多列去重
8. 跳过空表和临时文件
9. 导出 合并结果.xlsx
10. 输出成功、跳过、失败文件数量

## 暂时不做

1. 不支持 .xls
2. 不保留复杂格式、图表、宏
3. 不处理加密文件
4. 不做复杂 GUI
5. 不做多层文件夹递归扫描
6. 不处理百万行超大文件
