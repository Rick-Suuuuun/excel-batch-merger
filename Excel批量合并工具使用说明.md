# Excel批量合并工具使用说明

## 一、这个工具能做什么？

Excel批量合并工具可以把一个文件夹中的多个 Excel 文件合并成一个结果文件。

它可以帮助您：

- 一次合并多个 Excel 文件
- 自动跳过空表
- 自动跳过表头不一致的文件
- 在结果中标明每行数据来自哪个文件
- 按订单号、手机号等内容删除重复数据
- 保留原始 Excel 文件，不会对原文件进行修改

## 二、使用前需要准备什么？

请先把需要合并的 Excel 文件放在同一个文件夹中。

目前只支持文件名以 `.xlsx` 结尾的 Excel 文件，不支持 `.xls` 文件。

每个文件默认读取第一个工作表。需要合并的文件应使用相同的表头，例如：

```text
订单号、姓名、手机号、金额
```

## 三、如何打开软件？

找到“Excel批量合并工具”程序，双击打开。

打开后，您会看到：

- “选择文件夹”按钮
- 已选择文件夹的显示位置
- 去重字段输入框
- “开始合并”按钮
- 处理结果显示区域

## 四、如何选择 Excel 文件夹？

1. 点击“选择文件夹”。
2. 找到存放 Excel 文件的文件夹。
3. 选中文件夹并确认。
4. 软件界面会显示您选择的文件夹路径。

软件只读取所选文件夹中的 Excel 文件，不会继续查找里面的子文件夹。

## 五、如何填写去重字段？

去重字段就是用来判断两行数据是否重复的内容。

如果希望按照订单号去重，请输入：

```text
订单号
```

如果希望按照手机号去重，请输入：

```text
手机号
```

如果需要同时按照订单号和手机号判断重复，请输入：

```text
订单号,手机号
```

多个字段之间可以使用中文逗号或英文逗号。

字段名称必须和 Excel 第一行中的表头名称一致。如果填写了不存在的字段，软件会弹窗提示，请检查后重新填写。

发现重复数据时，软件会保留第一次出现的记录，删除后面重复的记录。

## 六、如果不需要去重怎么办？

如果不需要删除重复数据，请把去重字段输入框留空，直接点击“开始合并”。

软件会保留全部合并数据。

## 七、如何开始合并？

确认文件夹和去重字段后，点击“开始合并”。

处理完成后，界面会显示：

- 成功读取的文件数量
- 跳过的文件数量
- 读取失败的文件数量
- 合并前的数据行数
- 去重后的数据行数
- 删除的重复行数
- 结果文件的保存路径

## 八、结果文件保存在哪里？

处理完成后，请查看界面中的“输出文件路径”。这里会显示结果文件保存的完整位置。

结果文件通常保存在软件的 `output` 文件夹中。

文件名会根据去重方式自动生成，例如：

- 不去重：`合并结果_未去重.xlsx`
- 按手机号去重：`合并结果_按手机号去重.xlsx`
- 按订单号去重：`合并结果_按订单号去重.xlsx`
- 按订单号和手机号去重：`合并结果_按订单号_手机号去重.xlsx`

如果已经存在同名结果文件，软件会用新的结果覆盖它。覆盖前请先关闭已经打开的结果文件。

## 九、遇到空表或表头不一致会怎样？

空表不会参与合并，软件会自动跳过。

软件会根据大多数文件使用的表头确定标准表头。表头名称或顺序与标准表头不一致的文件，也不会参与合并。

这些文件不会被删除，也不会被修改。处理完成后，可以在界面中看到跳过文件的数量。

如果不同表头出现的次数相同，软件会提示您选择需要使用的标准表头。

## 十、原始文件会被修改吗？

不会。

软件只读取您选择的 Excel 文件，并另外生成一个新的结果文件。原始 Excel 文件的内容、名称和保存位置都不会改变。

## 十一、常见提示

### 提示“没有可读取的 .xlsx 文件”

请确认：

- 选择的文件夹中确实有 Excel 文件
- 文件名以 `.xlsx` 结尾
- 文件没有放在更深一层的子文件夹中

### 提示“没有可参与合并的 Excel 文件”

可能是文件都是空表，或者文件内容、表头不符合要求。请检查 Excel 第一行的表头。

### 提示“处理失败”

请先关闭正在打开的 Excel 文件，然后重新操作。如果仍然失败，请检查文件是否损坏或加密。

## 十二、目前的使用范围

- 只支持 `.xlsx` 文件
- 每个文件只读取第一个工作表
- 不读取子文件夹中的文件
- 不处理加密或损坏的 Excel 文件
- 不保留原文件中的复杂格式、图片、图表或宏
